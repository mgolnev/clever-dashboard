# Roadmap

| Этап | Содержание | Статус |
|------|-----------|--------|
| **MVP-1** | Битрикс через загрузку файла: импорт, нормализация, дашборд KPI/воронки/срезов + товарная аналитика, сравнение периодов | ✅ готово |
| **MVP-2** | Bitrix REST API (вебхук) + авто-обновление по расписанию (убрать ручную загрузку) | план |
| **MVP-3** | Яндекс Метрика: трафик, визиты, конверсия сайта, источники | план |
| **MVP-4** | Аппметрика: воронка приложения, единый отчёт app + web | план |

## Возможные улучшения MVP-1

- Несколько графиков динамики по дням внутри периода.
- Экспорт недельного отчёта (PDF/ссылка).
- Справочник товаров по `offer_id` из Битрикса (точные категории/бренды вместо
  эвристики по названию).
- Когортный анализ клиентов (новые/повторные).
